AspectJTM for Emacs

Version 1.1b2 released on Wednesday Dec 18, 2002.

This support for using Emacs to compile and run AspectJ programs and to view and navigate crosscutting structure in GNU Emacs 20.7, XEmacs 21.4.3 and JDEE 2.2.9. It is released under the GPL.

Relevant resources:

  • aspectj-mode (readme) only requires java-mode. The docs will say to install aspectj-mode.el using sample.emacs as a guide, and if you're a history buff, you might want to see the changes file.
  • AJDEE (readme) builds on JDEE. The docs will say to install aspectj-mode.el and ajdee.el. using sample.emacs and sample.prj as guides. This also has a changes file.
  • You can find this distribution on the AspectJForEmacs project page http://aspectj4emacs.sourceforge.net, where you can submit bugs on, or otherwise contribute to, AspectJForEmacs.
  • The AspectJ home page is at http://eclipse.org/aspectj. You'll need to download the appropriate version to use this support; also go here for language and compiler usage questions.
This project has not been tested against the 1.1 release. In particular, AspectJ 1.1 has not implemented javadoc (ajdoc) support, so you can't use it from emacs.
